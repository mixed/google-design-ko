텍스트 필드

텍스트 필드는 사용자에게 텍스트 입력과 텍스트 선택(잘라내기, 복사하기, 붙여넣기)과 자동 완성을 통한 데이터 찾기를 허용합니다.

텍스트 필드를 터치하면 커서가 놓여지고 자동으로 키보드가 표시됩니다. 텍스트 선택 디자인을 위해 패턴 > 선택을 확인해 보십시오.

차례

  • 형식
  • 한 줄 텍스트 필드
  • 플로팅 레이블
  • 여러 줄 텍스트 필드
  • 전체 너비 텍스트 필드
  • 글자 카운터
  • 자동 완성 텍스트 필드
  • 검색 필터

형식

텍스트 필드는 한 줄이거나, 스크롤이 있거나 없거나, 또는 여러 줄일 수 있습니다. 이것은 또한 아이콘을 표시할 것입니다.

허용 가능한 문자

텍스트 필드의 글자(Type)는 필드 내에 허용된 문자의 종류가 무엇인지를 결정합니다. 가상 키보드 레이아웃은 자주 사용된 문자를 위해 최적화될 것입니다.

당신이 최적화 해야 할 보통의 입력 유형은 다음을 포함합니다:

  • 숫자: 전화번호, 신용카드 번호, 카드 비밀번호(PIN)
  • 텍스트: 고유명사, 사용자 이름, URL
  • 혼합된 형식: 이메일 주소, 거리(Street) 주소, 검색 질의

자동 대문자 사용

블루투스 기기, 사용자, 지문, 알람과 같이 어떤 것에 이름 붙이기 위해 의도된 입력은 기본적으로 모든 단어가 자동으로 대문자로 사용되어야 합니다.

그 외에는, 문자 메세지나 화면 잠금과 같이 입력 필드가 문장과 유사한 메세지를 위해 의도된 것이라면, 오직 그 첫번째 단어만 자동으로 대문자가 되어야 합니다.

밀도

마우스와 키보드가 주 입력 방법일 때, 치수는 밀집한 레이아웃을 수용하기 위해 축소될 것입니다.

한 줄 텍스트 필드

한 줄 텍스트 필드

한 줄 필드는 텍스트 입력 커서가 입력 필드의 오른쪽 가장자리에 도달할 때 자동으로 내용물을 왼쪽으로 스크롤합니다.

밝은 테마

  • 힌트와 입력 폰트: Roboto Regular 16sp
  • 타일 높이: 48dp
  • 텍스트 상단과 하단 패딩: 16dp
  • 텍스트 필드 디바이더 패딩: 8dp

  • 타일 높이: 48dp
  • 텍스트 상단과 하단 패딩: 16dp
  • 텍스트 필드 디바이더 패딩: 8dp

플로팅 레이블

플로팅 레이블

사용자가 텍스트 입력 필드를 사용할 때, 플로팅 인라인 레이블은 그 필드 위에 떠 있기 위해서 이동합니다.

밝은 테마

  • 힌트와 입력 폰트: Roboto Regular 16sp
  • 레이블 폰트: Roboto Regular 12sp
  • 타일 높이: 72dp
  • 텍스트 상단과 하단 패딩: 16dp
  • 텍스트 필드 디바이더 패딩: 8dp

여러 줄 텍스트 필드

여러 줄 텍스트 필드는 넘치는 텍스트를 위해 자동으로 새 줄로 줄 바꿈하고, 커서가 아래쪽 가장자리에 닿을 때 수직으로 스크롤합니다.

밝은 테마

  • 힌트와 입력 폰트: Roboto Regular 16sp
  • 레이블 폰트: Roboto Regular 12sp
  • 텍스트 상단과 하단 패딩: 16dp
  • 텍스트 필드 디바이더 패딩: 8dp
