텍스트 필드

텍스트 필드는 사용자에게 텍스트 입력, 텍스트 선택, 그리고 자동 완성을 통한 데이터 찾기를 허용합니다.

목차

  • 입력
  • 레이블
  • 스타일
  • 한 줄 텍스트 필드
  • 여러 줄 텍스트 필드
  • 전체 너비 텍스트 필드
  • 글자 카운터
  • 자동 완성 텍스트 필드
  • 검색 필터
  • 필수 필드

입력

텍스트 필드는 한 줄이거나, 스크롤이 있거나 없거나, 또는 여러 줄일 수 있습니다. 이것은 또한 아이콘을 표시할 것입니다.

입력 문자

텍스트 필드의 글자(Type)는 필드 내에 허용된 문자의 종류가 무엇인지를 결정합니다. 가상 키보드 레이아웃은 자주 사용된 문자를 위해 최적화될 것입니다.

당신이 최적화 해야 할 보통의 입력 유형은 다음을 포함합니다:

  • 숫자: 전화번호, 신용카드 번호, 카드 비밀번호(PIN)
  • 텍스트: 고유명사, 사용자 이름, URL
  • 혼합된 형식: 이메일 주소, 거리(Street) 주소, 검색 질의

자동 대문자

각 텍스트 필드에서 첫 번째 문자는 로케일에 의해서 요구된다면 각 문장의 첫 번째 문자 뿐만 아니라 적절한 곳에서 대문자가 되어야 합니다. 예를 들어, 이것은 특히 다음과 같은 입력 필드와 연관이 있습니다:

  • 블루투스 기기, 사용자, 지문, 또는 알람과 같이 어떤 것에 이름을 요청하는 경우
  • 텍스트 메세지와 같이 문장과 유사한 메세지가 들어 있는 경우

레이블

사용자가 텍스트 입력 필드와 관계를 맺을 때, 플로팅 인라인 레이블은 그 필드 위에 떠 있기 위해 이동합니다.

레이블 색

레이블의 색은 당신의 앱의 색상 팔레트를 반영하지만 적절한 대비율(Contrast Ratio)를 가져야 합니다.

밝은 테마

어두운 테마

명세

  • 힌트와 입력 폰트: Roboto Regular 16sp
  • 레이블 폰트: Roboto Regular 12sp
  • 타일 높이: 72dp
  • 텍스트 상단과 하단 패딩: 16dp
  • 텍스트 필드 디바이더 패딩: 8dp

스타일

밀도

마우스와 키보드가 주된 입력 방법일 때, 치수는 더 밀집한 레이아웃을 수용하기 위해 줄어들 것입니다.

색

텍스트 필드는 당신의 제품의 색상 팔레트를 반영해야 합니다. 다음과 같은 색 적용이 권장됩니다:

  • 텍스트 필드와 텍스트 색: 텍스트 필드와 그 텍스트 필드의 커서에 당신의 팔레트의 강조색 또는 대비되는 색을 적용하십시오.
  • 에러 상태: 빨간색 또는 오렌지색과 같은 따뜻한 색조를 가진 에러 상태를 위한 대비 색을 사용하십시오.

한 줄 텍스트 필드

한 줄 필드는 텍스트 입력 커서가 입력 필드의 오른쪽 가장자리에 도달할 때 자동으로 내용물을 왼쪽으로 스크롤합니다.

밝은 테마

당신이 선택한 색은 다신의 앱의 색상 팔레트를 반영해야 합니다.

  • 힌트와 입력 폰트: Roboto Regular 16sp
  • 타일 높이: 48dp
  • 텍스트 상단과 하단 패딩: 16dp
  • 텍스트 필드 디바이더 패딩: 8dp

  • 텍스트 필드 디바이더 패딩: 8dp

여러 줄 텍스트 필드

여러 줄 텍스트 필드는 넘치는 텍스트를 위해 자동으로 새 줄로 줄 바꿈하고, 커서가 아래쪽 가장자리에 닿을 때 수직으로 스크롤합니다.

밝은 테마

당신이 선택한 색은 당신의 UI의 색 구성을 반영해야 합니다.

  • 힌트와 입력 폰트: Roboto Regular 16sp
  • 레이블 폰트: Roboto Regular 12sp
  • 텍스트 상단과 하단 패딩: 16dp
  • 텍스트 필드 디바이더 패딩: 8dp

전체 너비 텍스트 필드

전체 너비 텍스트 필드는 더 심도 있는 작업을 위해 유용합니다.

  • 힌트와 입력 폰트: Roboto Regular 16sp
  • 텍스트를 위한 상단과 하단 패딩: 20dp

자동 완성 텍스트 필드

드롭다운에서 실시간 제안과 완성을 보여 주기 위해서 자동 완성 텍스트 필드를 사용하면, 사용자는 더 정확하고 효율적으로 정보를 입력할 수 있습니다.

자동 완성은 특히 제한적으로 텍스트 쓰기와 읽기가 가능한 사용자나 스펠링에 어려움을 겪는 사람, 특히 모국어가 아닌 언어를 사용하는 이들에게 가치가 있습니다. (어떤 사용자들은 그들의 자국어로 된 페이지가 존재하지 않기 때문에 영어로 인터넷을 탐색해야만 합니다.)

삽입된(Inset) 자동 완성

인라인 자동 완성

검색 필터

앱 바는 텍스트 입력 필드로 동작할 수 있습니다. 사용자가 입력한 대로, 내용물 하부에 필터링되고 정렬됩니다.

필수 필드

필드가 필수적이라는 것을 나타내기 위해서, 애스터리스크(Asterisk, *)를 그 필드 다음에 표시하십시오. 그 폼의 최하단에, 애스터리스크가 필수 필드(Required Field)를 나타낸다는 것을 설명하는 노트를 포함하십시오.

헬퍼 텍스트

헬퍼 텍스트(Helper Text), 또는 힌트 텍스트(Hint Text)는 텍스트 필드 아래에 보여져야 하며, 지속적으로 보여지거나 또는 초점을 얻었을 때만 보여져야 합니다.
